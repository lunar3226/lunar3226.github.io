# 马来西亚COVID-19流行病监测分析系统

## 系统简介

本系统是一个基于Streamlit开发的交互式数据可视化平台，用于展示马来西亚2021-2025年COVID-19疫情数据分析结果。

### 核心功能

| 模块 | 功能说明 |
|------|----------|
| 数据概览 | 展示数据集基本信息、时间范围、关键统计指标 |
| 疫情趋势分析 | 病例/死亡趋势、累计趋势、检测关联分析 |
| 疫苗接种分析 | 接种率与疫情关联分析 |
| 医疗系统压力 | 医疗资源利用情况、各州对比 |
| 年龄分布分析 | 疫苗接种年龄分布可视化 |

### 技术特点

-  纯Python开发，无需前端技术
-  交互式数据筛选（日期范围）
-  实时图表生成
-  响应式布局，适配不同屏幕
-  支持图表导出
-  科研风格界面，适合学术展示

## 安装指南

### 1. 环境要求

| 项目 | 版本要求 |
|------|----------|
| Python | 3.7 或更高版本 |
| 操作系统 | Windows / Linux / macOS |

### 2. 安装依赖

打开命令行，执行以下命令：

```bash
pip install -r requirements.txt
```

或手动安装：

```bash
pip install pandas matplotlib seaborn numpy streamlit
```

### 3. 验证安装

```bash
streamlit --version
```

如显示版本号则安装成功。

## 运行指南

### 方式一：本地运行

1. 打开命令行，进入项目目录：

```bash
cd "d:\马来西亚流行病监测模型"
```

2. 运行应用：

```bash
streamlit run app.py
```

3. 浏览器会自动打开 `http://localhost:8501`

### 方式二：指定端口运行

```bash
streamlit run app.py --server.port 8080
```

### 方式三：网络访问

```bash
streamlit run app.py --server.address 0.0.0.0
```

## 使用说明

### 1. 侧边栏配置

**时间范围选择**：
- 选择分析的起始日期和结束日期
- 默认范围：2021-01-01 至 2025-12-31

**模块导航**：
- 点击对应模块切换分析视图

### 2. 数据概览模块

- 查看数据集统计信息
- 查看数据时间范围
- 查看关键统计指标（累计病例、死亡、病死率）
- 了解数据来源说明

### 3. 疫情趋势分析模块

- **COVID-19总体趋势**：双Y轴折线图，展示新增病例和死亡数
- **累计趋势**：展示累计病例和死亡数的变化
- **检测关联**：散点图展示检测量与新增病例的关系

### 4. 疫苗接种分析模块

- **接种率vs病例**：双Y轴图展示接种率与新增病例的关系
- 分析结论：疫苗接种对控制疫情的作用

### 5. 医疗系统压力模块

- **资源利用**：堆叠面积图展示各州年度医疗资源使用情况
- **统计表格**：展示各州总床位使用量排名

### 6. 年龄分布分析模块

- **年龄覆盖**：堆叠条形图展示各年龄段接种覆盖情况
- 分析结论：不同年龄段的接种策略建议

## 数据说明

### 数据来源

马来西亚官方COVID-19公开数据集（GitHub）

### 数据类型

| 数据类型 | 文件名 | 说明 |
|----------|--------|------|
| 病例数据 | epidemic_cases_malaysia.csv | 全国每日新增病例 |
| 死亡数据 | epidemic_deaths_malaysia.csv | 全国每日新增死亡 |
| 检测数据 | epidemic_tests_malaysia.csv | RTK-Ag和PCR检测量 |
| 疫苗数据 | vaccination_vax_malaysia.csv | 每日疫苗接种量 |
| 医院数据 | epidemic_hospital.csv | 医院床位使用情况 |
| ICU数据 | epidemic_icu.csv | ICU床位使用情况 |
| PKRC数据 | epidemic_pkrc.csv | 低风险隔离中心数据 |
| 年龄数据 | vaccination_vax_demog_age.csv | 各年龄段接种分布 |
| 人口数据 | static_population.csv | 各州人口统计 |

### 数据时间范围

- **分析范围**：2021年1月1日 至 2025年12月31日
- **数据更新**：静态历史数据

## 常见问题

### Q1: 运行时提示"No module named 'streamlit'"

**解决方法**：安装streamlit
```bash
pip install streamlit
```

### Q2: 图表显示乱码

**原因**：系统缺少中文字体

**解决方法**：系统已配置默认字体，如仍有问题，可修改代码中的字体设置

### Q3: 浏览器无法打开

**原因**：端口被占用或防火墙阻止

**解决方法**：
1. 更换端口：`streamlit run app.py --server.port 8080`
2. 检查防火墙设置

### Q4: 数据加载失败

**原因**：数据文件路径不正确或文件缺失

**解决方法**：
1. 确认`data/`目录下包含所有CSV文件
2. 确认从项目根目录运行

### Q5: 页面加载缓慢

**原因**：数据量较大或网络问题

**解决方法**：
1. 缩小分析时间范围
2. 关闭其他占用内存的程序
